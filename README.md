# 🦠 COVID-19 India Dashboard

A simple Streamlit dashboard that displays COVID-19 India state-wise data using live data from the COVID-19 India CSV API.

## Features

* View state-wise COVID-19 statistics
* Confirmed, Recovered, and Death cases
* Recovery rate calculation
* Interactive state selection
* Data table view
* COVID-19 trend charts

## Technologies Used

* Python
* Streamlit
* Requests
* CSV

## Data Source

https://data.covid19india.org/csv/latest/states.csv

## Installation

```bash
pip install streamlit requests
```

## Run the Project

```bash
streamlit run app.py
```
